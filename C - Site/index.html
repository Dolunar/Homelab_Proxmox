What I've set up
🌐Self-hosted website✅ live

This very site runs on a Debian 12 LXC with Nginx, hosted on my home Proxmox. Publicly accessible via Cloudflare Tunnel — no need to open ports, even with an ISP router that blocks ports 80 and 443.

NginxDebian 12Cloudflare TunnelLXCdolunay.fr
🗄️NAS with TrueNAS Scale✅ live

12TB of storage with a ZFS pool, separate datasets (media, documents, backups, vms), SMB share accessible from Windows, automatic daily snapshots. Proxmox VMs are automatically backed up to it.

TrueNAS ScaleZFSSMBSnapshots12TB
🛡️Pi-hole + DNS over HTTPS✅ live

Local DNS server that blocks ads across the entire network without installing anything on each device. DNS requests are encrypted via dnscrypt-proxy to Cloudflare. Machines are accessible by name (.lan) instead of typing IPs manually.

Pi-holeDNS over HTTPSdnscrypt-proxyAd blocking.lan
🔒WireGuard VPN✅ live

Secure remote access to my entire network from anywhere in the world. Installed via PiVPN on a Debian 12 LXC. DNS points to Pi-hole — so even on 4G, ads are blocked and .lan names work.

WireGuardPiVPNLXCFreeboxVPN
💾Automatic backups✅ live

Robocopy + Windows Task Scheduler to automatically sync my folders to the NAS every night at 9pm. Documents, iCloud photos — everything lands on the NAS without me doing anything. Simple, native, reliable.

RobocopyWindows Task SchedulerSMBAutomated
📊Monitoring — coming soonplanned

Uptime Kuma to monitor that all my services are running and get alerted if something goes down. Grafana for metrics. That's the next step.
